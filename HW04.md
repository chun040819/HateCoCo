# Data Flow Diagram (DFD)

## 圖解

```mermaid
graph TD;
    subgraph 使用者端
        使用者 -->|訊息| LINE官方帳號;
        LINE官方帳號 -->|發送問題| 後端API;
    end
    
    subgraph 系統處理端
        後端API -->|分析問題| RASA/NLP引擎;
        RASA/NLP引擎 -->|處理結果| 自動回覆系統;
        自動回覆系統 -->|回覆訊息| 後端API;
    end
    
    subgraph 客服支援端
        後端API -->|無法處理的問題| 工單系統;
        工單系統 -->|分派工單| 客服人員;
        客服人員 -->|解決方案| 工單系統;
        工單系統 -->|回覆結果| 使用者;
    end
